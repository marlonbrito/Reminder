# Reminder
app para lerbrar o usuario de suas tarefas

## O que tem aqui?

* Notificação no Browser
* Import/Export de módulos
* Async/Await
* Tratamento de excessões
* JS Moderno (e puro)
* Trabalhar com timer
* Sistema de Eventos Pub/Sub
* App Desktop com Electron
* ... muito mais
